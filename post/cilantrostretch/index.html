I’m Back with Cilantro

After recording my initial timelapse of basil, I learned many things that would make the process much easier than the first time. Since my cilantro decided to finally pop through the soil, I decided I would use this knowledge going forward to make yet another timelapse of seedlings, but this time much easier. The first thing I did was used what I thought was a higher quality camera (Logitech c270 I believe). This ended up having minimal impact in my opinion. I look forward to getting an intervelometer for my Nikon D3100 (Thanks Ted!) in the future but for now, this will work. I mounted it somewhat jankily to my tripod and pointed it at my cilantro at about noon. I set the interval to 2 minutes. Looking back, shooting for a higher framerate and taking pictures more frequently would have made sense. Turns out these cilantro babies are pretty active! I also saved the pictures with a different naming scheme that made it WAY easier to use ffmpeg to assemble the timlapse video. I prefixed the images with an h, to represent that they are an herb, and then followed it by the image number padded on the left by 0’s. So, for instance, the first image name would be h0001.jpg. Using this naming scheme allowed me to use the wildcard notation on ffmpeg and running this command ffmpeg -framerate 10 -pattern_type glob -i "h*.jpg" -c:v libx264 -crf 0 output.mp4 (thanks StackOverflow!) allowed me to assemble a timelapse seemlessly. Below, you can find both the new, slightly modified code to do such as well as the video of the cilantro waving away!

Code

from cv2 import *
import time

# the index changed to 2 because I am no longer using a built in webcam!
cam = VideoCapture(2)

for i in range(0,500):
    s, img = cam.read()
    if s:
	fname = "h" + str(i).rjust(4, "0")
        imwrite("herbs/" + fname + ".jpg",img)
       
    time.sleep(120)
